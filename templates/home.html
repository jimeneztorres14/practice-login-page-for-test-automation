Test Automation Playground

Welcome! This project is a UI Testing Playground inspired by the well-known “the-internet” project. It contains 10 interactive pages designed to help you practice and strengthen your Selenium WebDriver automation skills using Python.

Click any card below to open an interactive test page. Each one focuses on a common UI automation scenario.
