Test Automation Playground

Welcome! This project is a small playground inspired by the-internet where you can practice UI test automation with Selenium and Java.

Use the cards below to open individual test pages. Each page is focused on a specific type of UI interaction (login, checkboxes, etc.) and can be automated independently.

Authentication

Classic username/password login form with positive and negative scenarios, error messages, and a protected success page.
